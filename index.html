이모티콘 기획 도우미

카카오톡 이모티콘 심사를 염두에 둔, 귀엽고 앙증맞은 캐릭터 프롬프트 생성 도구 (시니어용 큰 글씨)

0. 이미지 생성 툴 바로가기

아래 버튼을 누르면 새 탭에서 이미지 생성 사이트가 열립니다.

Google AI Studio ImageFX Canva Figma

사용설명서

  1. 1번에서 대분류 → 소분류 → 세분화(10종 중 1개)를 순서대로 선택합니다.
  2. 2번에서 이모티콘이 쓰일 상황 카테고리를 선택합니다. (기본감정, 직장생활, 리액션, 일상생활, 안전사고/경보 등)
  3. 3번의 “이모티콘용 프롬프트 9개 생성하기” 버튼을 누르면 캐릭터 외형(세분화) + 선택한 상황 카테고리를 조합해서 9개의 프롬프트가 생성됩니다.
  4. 마음에 안 들면 아래 “랜덤 상황 9개 재생성” 버튼을 눌러 상황만 다시 섞어서 9개를 새로 만듭니다.
  5. 각 프롬프트를 복사해서 Google AI Studio / ImageFX / Canva 등으로 가져가 이미지를 생성하면 됩니다.

※ “다시 보지 않기”를 체크하고 저장하면, 다음부터는 이 설명서가 아예 보이지 않습니다.

1. 캐릭터 선택

대분류(동물/사람/아이콘 등) → 소분류(강아지/고양이/학생/시니어 등) → 세분화(외형 10종 중 하나)를 순서대로 고르세요.

2. 상황 카테고리 선택

이모티콘이 쓰일 상황의 종류를 선택하세요. 선택한 카테고리 안에서 9개의 상황이 랜덤으로 뽑힙니다.

3. 이모티콘용 프롬프트 9개 생성

카카오톡에 승인 잘 날 수 있도록, 텍스트/말풍선 없이 귀엽고 앙증맞은 캐릭터 위주의 프롬프트를 자동으로 작성합니다.
